Sistema de Agendamento

Este projeto é um sistema de agendamento de clientes, feito em PHP, HTML e CSS, com banco de dados MySQL.

Funcionalidades

Adicionar clientes

Listar clientes

Excluir clientes

Agendar compromissos


Tecnologias

PHP

MySQL

HTML / CSS
